# Eight Queens Question

## Installation

Recommand using python version 3.11.5 and install package with command :

```
pip install -r requirements.txt
```

## 參數說明



| 變數 | 說明 
| --------- | --------- 
| queen_num | 皇后數量
| ans_num   | 總答案數量
| division_8 | 有八種相似答案的答案個數
| division_4 | 有四種相似答案的答案個數
| division_2 | 有二種相似答案的答案個數
